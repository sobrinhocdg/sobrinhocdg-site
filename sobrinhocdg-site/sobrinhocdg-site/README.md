# 🎤 sobrinhocdg | Site Oficial

Este site é feito para flexionar o trabalho do rapper sobrinhocdg. Ele conecta com Supabase para exibir streams, vídeos do YouTube, projetos e lore.

## 🛠️ Stack
- **Frontend**: HTML5, CSS3, Vanilla JS
- **Backend**: Supabase (PostgreSQL + Auth)
- **Deploy**: Vercel
- **Dev**: GitHub Codespaces

## 📁 Como usar nos Codespaces
1. Este repositório já contém todo o código.
2. As tabelas no Supabase devem ser criadas com os nomes: `musicas`, `videos_yt`, `projetos`, `lore`.
3. As credenciais já estão em `js/script.js`.
4. Subir o servidor com `npm start` ou deployar no Vercel.

## 🎯 O que cada tabela faz:
| Tabela | Descrição |
|--------|-----------|
| musicas | Músicas com cover, Spotify e contagem de streams |
| videos_yt | Vídeos do YouTube, com o `video_id` (ex: `dQw4w9WgXcQ`) |
| projetos | Projetos paralelos, coletivos ou negócios |
| lore | Histórias, temáticas, enfim, o que define a sua marca |

## 🚀 Deploy na Vercel
1. Acesse [vercel.com](https://vercel.com)
2. Importe este repositório
3. DONE.
